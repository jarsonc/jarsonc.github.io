FIRST Inspires Iowa

FIRST Inpsires Iowa is a website I contribute to in conjunction with a Garmin Engineer, Jearmie Vens. Although it is his project, I contribute to the website through consulation, and backend work. I focus on the live scoring system, which pushes live scoring for global access, as well as provide an archive of awards and scoring data. The website uses HTML for the front end, hosted through Google's Firebase. The scorewatcher uses a Java applet designed to automacially upload data as it is updated. The website and source code are both avaliable online as part of an attempt to demistify the scoring system of the FIRST Tech Challenge and provide live scoring and updates during Iowa based competitions.
